1. Select all text in the textarea above (Ctrl+A or Cmd+A).
2. Copy the text (Ctrl+C or Cmd+C).
3. Open your data/manga-data.json file in your code editor.
4. Replace the entire content of data/manga-data.json with the copied text (Ctrl+V or Cmd+V).
5. Save data/manga-data.json file.
6. Commit and push changes to your GitHub repository.
